你是否聽過 Web Components 這個網頁標準？ Lit 是個 Web Components 的框架，可以讓你使用 JavaScript 來打造自定義的 HTML 元素。

## 什麼是 Web Components

Web Components 是一組標準，用於創建可重用的 HTML 元素。它們可以包含 HTML、CSS 和 JavaScript，並且可以被其他網頁使用。
